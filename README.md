# Iris Detection Neural Network

## Project Overview
This project implements a neural network for iris flower classification based on the famous Iris dataset. It aims to accurately predict the species of iris flowers based on their features.

## Features
- Multi-class classification support
- Customizable neural network architecture
- Detailed training and evaluation metrics
- Data visualization tools
